# 🎓 Happy Ending Play School Management System

A web-based Play School Management System developed using Python and Django. This project helps manage students, teachers, admissions, attendance, marks, homework, and other school activities through separate Admin, Teacher, and Student portals.

---

## 📌 Project Description

The Happy Ending Play School Management System is designed to simplify the daily operations of a play school. It provides separate dashboards for Admin, Teachers, and Students, allowing efficient management of admissions, attendance, marks, homework, and communication.

---

## ✨ Features

- 👨‍💼 Admin Dashboard
- 👩‍🏫 Teacher Dashboard
- 👨‍🎓 Student Dashboard
- 📝 Online Admission Form
- ✅ Attendance Management
- 📊 Student Marks Management
- 📚 Homework Management
- 👩‍🏫 Teacher Recruitment
- 📸 School Gallery
- 📩 Email Notification
- 🔐 Secure Login System

## ▶️ Installation Steps

### Clone the Repository

```bash
git clone https://github.com/jananidj1805-collab/HappyEndingPlaySchool.git
```

### Go to Project Folder

```bash
cd HappyEndingPlaySchool
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux / Mac

```bash
source venv/bin/activate
```

### Install Requirements

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py migrate
```

### Start Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
